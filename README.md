## 📂 Projeto: SSH Client em Python

### 📌 Descrição

O arquivo `ssh_cmd.py` implementa um cliente SSH utilizando a biblioteca **Paramiko**.

A ideia surgiu para atender situações em que você precisa acessar remotamente um servidor via SSH, mas não possui um cliente SSH instalado na máquina. Se houver uma instalação do Python disponível, a ferramenta pode ser utilizada como alternativa para estabelecer a conexão.

### ⚠️ Observação

Esta ferramenta foi desenvolvida para fins educacionais, laboratoriais e de administração de sistemas em ambientes autorizados.

---

## 🔧 Requisitos

Antes de executar o script, certifique-se de possuir:

* Python 3 instalado
* Biblioteca Paramiko instalada

Instalação do Paramiko:

```bash
pip install paramiko
```
